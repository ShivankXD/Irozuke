# 🎨 Irozuke | 色付け
### *Geometry in Motion. Color by Intelligence.*

<img width="520" height="500" alt="Irozuke_logo2" src="https://github.com/user-attachments/assets/606f5409-58a2-4b63-a6b0-ac4c82b18e87" />


**Irozuke** is an automated manga colorization platform. By leveraging advanced Computer Vision techniques, it identifies line art structures and applies consistent, context-aware color palettes to monochrome panels.

## ⚡ Tech Stack
- **Framework:** PyTorch
- **Architecture:** [e.g., Pix2Pix / Stable Diffusion Fine-tune]
- **Preprocessing:** OpenCV / line-extraction

## 🚀 Getting Started
1. Clone: `git clone https://github.com/shivankxd/irozuke.git`
2. Install: `pip install -r requirements.txt`
3. Run: `python src/inference.py --input sample.jpg`

## 🛠 Features
- [ ] Automatic line art detection
- [ ] Reference-based color styling
- [ ] Batch processing for entire chapters
